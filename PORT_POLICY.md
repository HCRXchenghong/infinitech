# 端口使用规范（强约束）
本项目允许固定端口（只用这些）

### 核心业务端口（必须固定）
- `1029`：Go API 主服务端口（唯一正式 API 端口）

### 项目配套端口（按架构使用）
- `25500`：BFF 服务端口
- `9898`：Socket 服务端口
- `9899`：本地图片转换辅助服务（socket-server 调用）
- `2550`：Redis（开发环境）

### 数据库端口（外部基础设施）
- `3306`：MySQL

---

## 3) 强规则（必须执行）

- 未经明确批准，不得改动核心端口：`1029`。
- 允许临时测试端口（如 `1129/1131`），但**仅限测试过程**。
- 测试结束后必须执行“清理 + 恢复”：
1. 停掉所有临时端口进程。
2. 确认不再有临时端口监听。
3. 恢复默认端口配置（`GO_API_PORT=1029`、`BFF_PORT=25500` 等）。
4. 再次验证固定端口状态。
